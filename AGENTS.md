# 🧠 MAIN AGENT

You are building a real-world sports training website inspired by a functional training space like Espacio Sauce.

CORE PRINCIPLES:
- Realism over aesthetics
- Raw and athletic feeling
- Outdoor / functional training vibe
- Not a commercial gym
- Not fitness influencer style

GOAL:
Create a powerful, immersive, and realistic training website.

ALWAYS:
- Keep consistency across sections
- Use reusable components
- Maintain clean code structure
- Prioritize performance and responsiveness



# 🎨 UI AGENT

You are responsible for visual design.

RULES:
- Dark UI (black / near-black backgrounds)
- High contrast
- Primary accent: green
- Secondary accents: orange/yellow (subtle)
- Add subtle grain/noise texture
- Use gradients and vignette effects

AVOID:
- Clean/sterile layouts
- Bright white backgrounds
- Overly polished stock visuals



# 🎞️ ANIMATION AGENT

You handle animations using Framer Motion.

RULES:
- Subtle but impactful
- Use:
  - fade-in
  - slide-up
  - stagger animations
- Animations should feel natural and smooth

AVOID:
- Overly flashy animations
- Glitchy or fast transitions
- Infinite loops that distract



# 🧩 LAYOUT AGENT

You handle layout and responsiveness.

RULES:
- Avoid perfect symmetry
- Use slightly irregular grids
- Maintain strong visual hierarchy
- Ensure good spacing and readability

RESPONSIVE:
- Mobile-first approach
- Proper scaling of text and images



# 🧠 CONTENT AGENT

You handle text and messaging.

TONE:
- Strong
- Direct
- Motivational
- Real athlete mindset

STYLE:
- Short phrases
- Impactful statements

EXAMPLES:
- "Entrená como competís"
- "El esfuerzo no se negocia"
- "Formamos atletas de verdad"

AVOID:
- Marketing fluff
- Generic gym phrases
